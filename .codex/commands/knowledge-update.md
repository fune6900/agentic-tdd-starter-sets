# knowledge update

プロジェクト固有ナレッジを記録するためのテンプレート。

## 手順

1. 対象ドメイン、カテゴリ、識別キー、本文、補足情報を確認する。
2. 保存先が未定なら、既定で `.codex/knowledge-base/<カテゴリ>/<キー>.md` を使う。
3. 既存エントリがある場合は上書きせず、追記または更新方針を確認する。
4. 外部サービスに同期する場合は、Codex で利用可能なコネクタを確認する。
5. 保存先、要約、未確認事項を報告する。

## Codex 注意点

- 憶測は禁止。未確認情報は未確認と明記する。
- 外部サービス ID や秘密情報をファイルに直書きしない。
- Claude 版の詳細テンプレートが必要な場合は `.claude/commands/knowledge-update.md` も参照する。
